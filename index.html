Mini jeu vidéo en C++

As part of my studies at the IUT of Paris-Saclay, I developed a small video game in C++ from scratch over several months. The project involved designing gameplay mechanics such as avoiding enemies and collecting coins, while progressively adding new features. This experience allowed me to work on object-oriented programming, game logic, event handling, and code structure, while strengthening my problem-solving and debugging skills.

CodeJam

During an event called the "Code Game Jam", we created a video game in a group of 6, coded in C# with Unity. It taught me how to work in a team and helped me develop new skills, such as C# and game development with Unity.

WebSite ~Nemoris

From brand identity to the final launch, we had to designed and built this website for a fictional travel agency. I conceptualized the travel offers and developed a fully functional, responsive interface. This project demonstrates my ability to handle a web project from scratch with both creativity and technical rigor.

Python games

I developed several mini-games in Python, including a Hangman game and a "Towers of Hanoi" simulation. These projects focused on mastering list manipulation, stacks, and recursion logic. It was a great way to strengthen my algorithmic thinking and code organization skills.

Wireframe for a music app

Using Penpot, I created an interactive high-fidelity mockup for a music streaming application. The project included designing features like "Favorites" and playback controls. By building animated prototypes, I learned how to design intuitive user flows and prioritize user-centered design principles.
